# TP Capstone – Projet Final

Ce dépôt contient le projet **Capstone** réalisé dans le cadre du cours sur MLIAEdu.  
Il s’agit d’un projet de synthèse qui met en pratique l’ensemble des notions étudiées au fil des TP précédents.

---

## 🎯 Objectifs du projet
- Mobiliser les compétences acquises en **Hibernate**, **JPA** et optimisation des performances.
- Concevoir une application complète intégrant une base de données relationnelle.
- Expérimenter les stratégies de cache, de préchargement et de mapping.
- Mettre en place des tests et benchmarks pour valider les choix techniques.
- Produire une documentation claire et un schéma d’architecture.

---

## 📂 Structure du projet
- `src/main/java` : Code source des entités, services et logique métier.
- `src/main/resources` : Fichiers de configuration (Hibernate, cache, base de données).
- `src/test/java` : Tests unitaires et benchmarks de performance.
- `docs/` : Documentation et diagrammes (draw.io).
- `pom.xml` : Dépendances Maven.

---

## ⚙️ Prérequis
- **Java 17+**
- **Maven 3.8+**
- Base de données (H2 pour les tests, PostgreSQL/MySQL pour la production)
- Hibernate 6.x
- EhCache / Infinispan
